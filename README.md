# Движок динамической фильтрации документов в 1С

Высокопроизводительный инструмент для платформы 1С:Предприятие 8.3, предназначенный для гибкого динамического отбора документов на основе анализа данных табличных частей и истории изменений их статусов.

cf - уже готовая среда программы с обработкой и созданными прикладными объектами

epf - сама обработка
